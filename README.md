# 🏦 Loan Approval Prediction using Machine Learning

A complete End-to-End Machine Learning web application that predicts whether a loan application is **Approved** or **Not Approved** based on an applicant's financial information.

## 🌐 Live Demo

🚀 **Live Application:** https://loan-approval-prediction-ml-akif7.onrender.com

---

# 📌 Project Overview

This project uses a Machine Learning classification model to predict loan approval based on applicant details.

The application is built using **Python**, **Flask**, **Scikit-learn**, **HTML**, and **CSS**, and is deployed on **Render**.

Users simply enter their information through an interactive web interface, and the model predicts whether the loan is likely to be approved.

# 📊 Input Features

The model uses the following features:

- Age
- Annual Income
- Credit Score
- Loan Amount
- Years Employed

---

# 🎯 Prediction Output

The model predicts one of two classes:

- ✅ Loan Approved
- ❌ Loan Not Approved
